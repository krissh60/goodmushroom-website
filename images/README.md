# Product Imagery

The HTML references these files. Run `bash download.sh` to populate them with stock photography from Pexels (free commercial use, no attribution required).

| Filename | Used on | Replace with |
|---|---|---|
| `hero.jpg` (1800w) | Homepage hero bg | Farm / harvest shot, wide landscape |
| `about.jpg` (1200w) | Homepage about section | Farm close-up or team/facility |
| `og-image.jpg` (1200×630) | Social sharing | Branded hero crop |
| `cordyceps.jpg` | Products | Orange Cordyceps militaris |
| `shiitake.jpg` | Products | Brown-capped Shiitake on wood |
| `lions-mane.jpg` | Products | White pom-pom Lion's Mane |
| `oyster.jpg` | Products | Pearl/blue oyster cluster |
| `reishi.jpg` | Products | Red/brown Reishi fruiting body |
| `button.jpg` | Products | White button / crimini |
| `extract.jpg` | Products | Extract powder, bottles, or capsules |
| `herbs.jpg` | Products | Ayurvedic herbs (ashwagandha, tulsi) |

**If an image is missing**, the HTML falls back to the existing emoji icon + branded gradient so the page still looks complete.

**Swap in real product photos later:** just drop a JPG with the same filename into this folder and rebuild — no code change needed.
